# Fawared - رخام صناعي فاخر

موقع إلكتروني حديث وعصري لشركة فوارد لعرض وتسويق الرخام الصناعي الفاخر.

## المميزات

- تصميم عربي RTL بالكامل
- واجهة مستخدم عصرية وأنيقة
- عرض ديناميكي للمنتجات
- تصميم متجاوب يعمل على جميع الأجهزة
- تحميل سريع وأداء ممتاز
- سهولة النشر على GitHub Pages

## هيكل المشروع

```
fawared-site/
├── index.html          # الصفحة الرئيسية
├── styles.css          # الأنماط والتصميم
├── script.js           # الوظائف والتفاعلات
├── images/             # الصور والرسومات
│   ├── logo.png        # شعار الشركة
│   ├── hero-bg.jpg     # خلفية القسم الرئيسي
│   ├── F06.jpg         # صور المنتجات
│   ├── F07.jpg
│   └── ...
├── products/           # بيانات المنتجات
│   └── products.json   # ملف بيانات المنتجات
└── README.md           # هذا الملف
```

## التقنيات المستخدمة

- HTML5 مع دعم كامل للغة العربية و RTL
- CSS3 مع تصميم متجاوب وحديث
- JavaScript Vanilla ES6+
- خط Cairo من Google Fonts
- تصميم متجاوب (Mobile-First)

## النشر على GitHub Pages

### الطريقة الأولى: عبر واجهة GitHub

1. أنشئ مستودع جديد باسم `username.github.io`
2. ارفع جميع ملفات المشروع
3. انتظر بضع دقائق ليتم النشر
4. زر `https://username.github.io`

### الطريقة الثانية: عبر GitHub Desktop

1. استنساخ المستودع محلياً
2. أضف جميع ملفات المشروع
3. قم بعمل commit و push
4. انتظر النشر التلقائي

### الطريقة الثالثة: عبر سطر الأوامر

```bash
git clone https://github.com/username/username.github.io.git
cd username.github.io
# انسخ جميع ملفات المشروع هنا
git add .
git commit -m "Initial commit - Fawared website"
git push origin main
```

## التعديلات المطلوبة قبل النشر

### 1. استبدال الصور النائبة
- أضف شعار الشركة الحقيقي في `images/logo.png`
- أضف خلفية احترافية في `images/hero-bg.jpg`
- أضف صور المنتجات الفعلية من F06 إلى F30

### 2. تحديث معلومات التواصل
في `index.html` استبدل:
- `PLACEHOLDER_PHONE` برقم الهاتف الفعلي
- `PLACEHOLDER_EMAIL` بالبريد الإلكتروني الفعلي  
- `PLACEHOLDER_WHATSAPP` برقم الواتساب الفعلي
- `PLACEHOLDER_LOCATION` بالعنوان الفعلي

### 3. تعديل بيانات المنتجات
في `products/products.json` يمكنك:
- تعديل أسماء المنتجات
- تغيير الأوصاف
- إضافة أو إزالة منتجات
