# Favicon Instructions

## Recommended Files
- `favicon.png` - 32x32px (standard)
- `favicon-16x16.png` - 16x16px (small icons)
- `favicon-32x32.png` - 32x32px (standard)
- `apple-touch-icon.png` - 180x180px (iOS home screen)

## How to Create
1. Take your logo
2. Resize to square format
3. Save as PNG with transparent background
4. Place in images/ folder

## Browser Support
- Chrome, Firefox, Safari: favicon.png
- iOS: apple-touch-icon.png
- Windows: favicon.ico (if needed)
