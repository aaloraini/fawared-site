## Image Specifications

### Technical Requirements
- Format: JPG or PNG
- Compression: Optimized for web (balance quality and file size)
- Color Profile: sRGB
- Resolution: 72 DPI (web standard)

### Style Guidelines
- Professional lighting
- Clean background
- Consistent angles and distances
- Show marble texture and patterns clearly
- Include scale reference if possible

## Image Optimization

Before deployment, optimize images using:
- TinyPNG (https://tinypng.com/)
- ImageOptim (Mac)
- Adobe Photoshop "Save for Web"

Target file sizes:
- Logo: < 20KB
- Hero background: < 500KB
- Product images: < 200KB each
